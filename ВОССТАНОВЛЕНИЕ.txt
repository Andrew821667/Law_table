===========================================
ИНСТРУКЦИЯ ПО ВОССТАНОВЛЕНИЮ - 5 МИНУТ
===========================================

ШАГ 1: Откройте Apps Script
----------------------------
1. Откройте Google Sheets таблицу
2. Расширения → Apps Script

ШАГ 2: Удалите старые файлы (если есть)
----------------------------------------
В редакторе Apps Script слева видите список файлов:
- Наведите на каждый файл
- Нажмите три точки → Удалить
- Удалите ВСЕ файлы кроме appsscript.json

ШАГ 3: Создайте новые файлы
----------------------------
ВАЖНЫЕ ФАЙЛЫ (ОБЯЗАТЕЛЬНО!):

1. Main.gs
2. Config.gs
3. UserManager.gs
4. AppLogger.gs
5. LegalWorkflowManager.gs
6. HearingNotifier.gs
7. TelegramNotifier.gs
8. TelegramBot.gs
9. CaseManager.gs
10. ClientDatabase.gs
11. TimeTracker.gs
12. TelegramCleanup.gs

Для каждого файла:
1. Нажмите + → Скрипт
2. Переименуйте в имя файла (например Main.gs)
3. Скопируйте содержимое из папки src/
4. Вставьте в редактор
5. Сохраните (Ctrl+S)

ШАГ 4: Проверка
---------------
1. Все файлы созданы?
2. В Main.gs есть функция onOpen()?
3. Нажмите Ctrl+S (сохранить все)
4. Закройте редактор
5. Вернитесь в таблицу
6. Нажмите F5 (обновить)

Меню "⚖️ Судебные дела" должно появиться!
